# StellarForge Docs

## Getting Started

Each template in `templates/` is self-contained. Pick the one closest to your use case, copy it, and follow its README.

## Templates

| Template | Description |
|---|---|
| [payment-node](../templates/payment-node) | Send XLM between accounts |
| [token-issuer](../templates/token-issuer) | Issue a custom Stellar asset |
| [soroban-counter](../templates/soroban-counter) | Minimal Soroban smart contract |
